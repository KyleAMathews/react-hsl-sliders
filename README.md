# React HSL Sliders

## Getting Started

```bash
npm install react-hsl-sliders
```

## Props

- value (hex color string)
- onChange (function)
- inline (boolean)
- tabIndex (number)
- hideValues (boolean)
- classNames (object)
  - container (string)
  - rangeGroup (string)
  - rangeLabel (string)
  - range (string)
